http://job4j.ru/
Целью проекта является разработка сайта объявлений.
В объявлении присутствует: описание, марка машины, тип кузова, фото.
Объявления можно добавлять/редактировать/удалять.

### Требования к окружению:
Java 17;\
PostgreSQL 14;\
Apache Maven 3.8.4
